[**Design Tokens**](https://doc.wikimedia.org/codex/latest/design-tokens/overview.html) worden gebruikt als "developer hand-off" voor de overdracht van
design elementen naar development. Aangezien W3 hier (nog) geen standaard voor heeft, kunnen
deze naar believe worden geformuleerd.

Onderstaande Design Tokens dienen als input voor de [**Create Atomic Resources**](https://www.npmjs.com/package/create-atomic-resources) package
dat gebruikt maakt van de [**Atomic Design principes**](https://atomicdesign.bradfrost.com).
Deze principes zijn in dit project geimplementeerd via het [**atomic-bomb**](https://www.npmjs.com/package/atomic-bomb) package.

```json
{ "paste":  "token-here"}
```
